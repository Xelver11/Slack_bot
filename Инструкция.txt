1.Зарегистрироваться в Slack
2.Создать workspace
3.Создать три открытых канала “test1”, ”test2”, “test3”
4.Создать приложение (бота) для отправки сообщений из json-файла в каналы (https://api.slack.com/)
5. В Virtual studio code создать bot.py
6. Ввести код из файла "бот код" (или открыть из приложенного файла одноименный)
7. Открыть в Virtual studio code .env в котором указан токен бота
8. Запустить код
9. Посмотреть в в слаке каналы тест1, тест2, тест3.
